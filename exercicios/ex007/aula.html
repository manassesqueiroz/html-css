Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em destaque usando a tag < b >(não semantica).

Nesta frase temos um termo em destaque usando a tag < strong >(semantico).

itálico / enfáse

Nesta frase temos um termo em enfáse usando a tag < i >(não semantica).

Nesta frase temos um termo em enfáse usanda a tag < em > (semantica).

Texto marcados

Podemos criar textos marcados uzando a tag < mark >.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Libero illum saepe tenetur magnam vel nostrum minus fuga quibusdam dolores? Nesciunt eligendi qui sequi blanditiis quas vero in quasi sint necessitatibus.

texto grande e texto pequeno

A tag < big > se tornol obsoleta mas a tag < small > ainda pode ser uzada pra ter letras pequenas.

Texto deletado

Podemos marcar um texto excluido com a tag < del > para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto inserido com a tag < ins > para dar enfáse e indicar que ele foi insirido.

Tambem existe a tag < u > mas não e indicada (não semantica).

Texto Sobrescrito

Usamos a tag < sup > : exemplo X2+4

Texto Subscrito

Usamos a tag < sub > : exemplo H2o
